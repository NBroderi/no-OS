Arm\textquotesingle{}s Platform Security Architecture (PSA) is a holistic set of threat models, security analyses, hardware and firmware architecture specifications, an open source firmware reference implementation, and an independent evaluation and certification scheme. PSA provides a recipe, based on industry best practice, that allows security to be consistently designed in, at both a hardware and firmware level. The PSA Cryptographic API (Crypto API) is an important PSA component that provides a portable interface to cryptographic operations on a wide range of hardware\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Overview}{Overview}}\label{index_autotoc_md1}
The PSA Crypto API provides a standardized interface for performing cryptographic operations, ensuring consistency and security across different platforms and implementations.\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Key Features}{Key Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Secure Key Management}}\+: Create, import, export, and manage cryptographic keys securely.
\item {\bfseries{Cryptographic Operations}}\+: Perform encryption, decryption, hashing, MAC, and digital signature operations.
\item {\bfseries{Algorithm Agility}}\+: Support for a wide range of cryptographic algorithms.
\item {\bfseries{Platform Independence}}\+: Consistent API behavior across different hardware and software platforms.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysubsection{\texorpdfstring{Modules}{Modules}}\label{index_autotoc_md3}
The PSA Crypto API is divided into several modules, each responsible for different aspects of cryptography\+:


\begin{DoxyItemize}
\item {\bfseries{Key Management}}\+: Functions for managing cryptographic keys.
\item {\bfseries{Message Digests}}\+: Functions for computing hash values.
\item {\bfseries{Message Authentication Codes (MAC)}}\+: Functions for generating and verifying MACs.
\item {\bfseries{Ciphers}}\+: Functions for encryption and decryption.
\item {\bfseries{Digital Signatures}}\+: Functions for generating and verifying digital signatures.
\item {\bfseries{Asymmetric Key Operations}}\+: Functions for asymmetric encryption and decryption.
\item {\bfseries{Key Derivation}}\+: Functions for deriving keys from existing keys or passwords (Not supported). 
\end{DoxyItemize}
