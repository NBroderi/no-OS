Example Applications

    USS (Legacy MAXQ10xx) API Examples

    • USS_simple_test_app
      		•  Demonstrates the communication with the security manager (either the MAXQ10xx Internal Emulation or an external MAXQ10xx component). It should be run first to generate a filesystem.
    • USS_provisioning
      		•  Provisions the necessary keys for TLS and other crypto examples.
    • USS_tls_freertos_lwip_adin1110
      		•  Demonstration of how to set up and test secure connections using TLS v1.2 and v1.3
    • USS_encryption_test_app
      		•  Demonstrates encryption and decryption of randomly generated data
    • USS_signature_gen_and_verify
      		•  Demonstrates how to generate a signature of random data and how to verify the signature
    • USS_tamper_test_app
      		•  Demonstrates use of the MXQ_Tamper API call

    Platform Security Architecture (PSA) Examples

    • USS_PSA_AEAD
      		•  This example shows the usage of PSA AEAD cipher APIs and runs some AEAD tests.
    • USS_PSA_hashing
      		•  This example takes a predefined set of data vectors and computes a hash value from those data vectors by using PSA Hash APIs.
    • USS_PSA_key_mgmt
      		•  This example shows the usage of PSA key management APIs.
    • USS_PSA_mac
      		•  This example shows the usage of PSA MAC APIs and runs some MAC tests.
    • USS_PSA_signature
      		•  Tests the hashing, signature and signature verification capabilities of the PSA API.
    • USS_PSA_unauth
      		•  This example shows the usage of PSA unauthenticated cipher APIs and runs some tests.
