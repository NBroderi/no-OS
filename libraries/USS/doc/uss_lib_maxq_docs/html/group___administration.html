◆ MXQ_ResetFactory()

mxq_err_t __API__ MXQ_ResetFactory ( void )

Performs a Factory Reset

Note
Only available on debug parts
Returns
Execution status.
Return values
MXQ_OKSuccess.
Precondition
MXQ_Module_Init() must have been called before.
Postcondition
All security conditions are disabled.
All pending operations (message digest, cipher, mac) are aborted
All internal context is reset (TLS session keys, Secure channel session keys, Internally stored randoms or digests)
All objects are deleted. Object 0x1000 (Key importation key) is reset to its default value. Object 0x1001 (Adminsitrator key is reset to its defautl value.
